# Medical Insurance Price Prediction 

A machine learning project that predicts insurance charges based on age, BMI, children, smoking status, gender, and region.

##  Features
- Clean data preprocessing pipeline
- Exploratory Data Analysis with visualizations
- Multiple regression models: Linear, Random Forest, XGBoost
- Auto-comparison of performance metrics (MAE, RMSE, R²)
- Feature importance visualization

##  Tech Stack
Python • Pandas • Scikit-learn • XGBoost • Matplotlib • Seaborn

##  How to Run
1. Place your `insurance.csv` dataset in the same folder.
2. Run:
   ```bash
   python insurance_price_prediction.py
